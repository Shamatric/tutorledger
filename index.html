Connect Google Sheets

Your data syncs across all devices via a Google Sheet you own. One-time setup, ~5 minutes.

1Create a Google Cloud project

Go to console.cloud.google.com → create a new project (e.g. "TutorLedger").

2Enable Google Sheets API

APIs & Services → Library → search "Google Sheets API" → Enable.

3Create an API Key

Credentials → Create Credentials → API Key. Copy it here.

4Create OAuth 2.0 Client ID

Credentials → Create Credentials → OAuth client ID → Web application.
Under Authorised JavaScript origins add the URL you host this file on (e.g. your GitHub Pages URL or http://localhost).

5Create a blank Google Sheet & paste its ID

Go to sheets.google.com, make a new blank sheet. The ID is the long string between /d/ and /edit in the URL.

docs.google.com/spreadsheets/d/PASTE_THIS_PART/edit
